# التوقيت السواحلي الديناميكي - ليبيا 🇱🇾 (Dynamic Swahili Time)

تطبيق ويب ذو صفحة واحدة (SPA) يحسب ويعرض "التوقيت السواحلي" الديناميكي بناءً على الموقع الجغرافي وحركة الشمس الحقيقية (شروق وغروب). التطبيق يدعم حالياً: طبرق، بنغازي، وطرابلس.

## 📂 هيكل المشروع
- `index.html`: البنية الأساسية للواجهة (يدعم Tailwind CSS عبر CDN).
- `css/style.css`: المتغيرات اللونية (Theme) والتنسيقات المخصصة والانتقالات.
- `js/app.js`: المنطق البرمجي، حسابات الوقت النسبي، والتواصل مع واجهة برمجة التطبيقات (API).

## 🚀 كيفية التشغيل (Setup)
1. قم بتحميل أو استنساخ (Clone) المستودع إلى جهازك.
2. لا توجد حاجة لتثبيت أي حزم (Zero-build process).
3. فقط قم بفتح ملف `index.html` في أي متصفح حديث، وسيعمل التطبيق فوراً.

## 🌗 تبديل المظاهر (Light & Dark Modes)
تم بناء محرك المظهر (Theme Engine) ليعمل **تلقائياً**. 
- يعتمد التبديل على موقع الشمس الفعلي في المدينة المحددة.
- إذا كان الوقت الحالي بين الشروق والغروب، يتم تفعيل المظهر النهاري (Light Stone).
- إذا كان الوقت بعد الغروب أو قبل الشروق، يقوم التطبيق بإضافة الكلاس `.theme-night` إلى عنصر `<body>` لتفعيل المظهر الليلي (Dark Stone) بسلاسة.

## 🌍 كيفية إضافة مدينة جديدة أو API مستقبلاً
لإضافة مدينة جديدة، لا تحتاج لتعديل هيكل HTML. كل ما عليك فعله هو إضافة بيانات المدينة في ملف `js/app.js` داخل كائن `cities`:

```javascript
const cities = {
    // ... المدن الحالية
    misrata: { 
        name: "مصراتة", 
        lat: "32.3754", 
        lng: "15.0925" 
    }
};
