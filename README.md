## 🌙 ما هو التوقيت السواحلي / المغربي؟

التوقيت السواحلي (أو الغروبي / الزوالي) نظامٌ تقليدي عريق لا يزال حياً في مناطق من ليبيا واليمن وشرق أفريقيا. يختلف جوهرياً عن الساعة الأوروبية التي تبدأ يومها من منتصف الليل.

| الساعة المدنية | التوقيت السواحلي التكيفي |
|:-:|:-:|
| يبدأ اليوم عند 00:00 (منتصف الليل) | يبدأ عدّاد النهار عند الشروق الفعلي (الساعة 1 نهاراً) |
| ساعة ثابتة = 60 دقيقة دائماً | ساعة قياسية (60 دقيقة) لكنها تُحسب من الشروق أو الغروب |
| مرجعها توقيت غرينتش | مرجعها حركة الشمس الحقيقية في مدينتك |
| 24 ساعة دائماً في اليوم | عدد الساعات يتغيّر: أقل في الشتاء وأكثر في الصيف |
| لا علاقة له بالطبيعة | يُزامن حياتك مع الشروق والغروب |

> **مثال:** إذا شرقت الشمس في طبرق عند 6:02 صباحاً، فإن السابعة مساءً بالتوقيت المدني
> تُقابل الساعة **الثالثة عشرة من النهار** بالتوقيت السواحلي — لأن ثلاث عشرة ساعة قياسية
> (60 دقيقة كل منها) مرّت منذ الشروق في يوم صيفي طويل.

## ✨ المزايا

### 🕐 الجوهر الحسابي
- **ساعة قياسية تكيفية** — يبدأ العدّاد بالساعة 1 عند لحظة الشروق (للنهار) أو الغروب (لليل)، ويعكس مجموعه الكلي الطول الحقيقي لنهارك أو ليلك في مدينتك — قد يتجاوز 12 في الصيف أو يقلّ عنها في الشتاء
- **بيانات فلكية حية** — شروق وغروب دقيق من [`Open-Meteo API`](https://open-meteo.com) مع بيانات الأمس واليوم وغداً في طلب واحد
- **كاش ذكي** — `localStorage` بمدة صلاحية 6 ساعات يقلّل الطلبات ويُحسّن الأداء
- **حماية من السباق** — عدّاد الجيل (generation counter) يمنع تداخل طلبات تبديل المدن السريع

### 🌆 الواجهة الديناميكية
- **سماء حية** — تدرّج لوني يتغير كل ثانية: ذهبي الفجر → أزرق الصباح → أبيض الظهر → ذهبي العصر → برتقالي الغروب → كحلي الليل
- **ثلاثة أثواب تلقائية** — نهاري / ذهبي (45 دقيقة حول الشروق والغروب) / ليلي
- **قوس شمسي متحرك** — شمسٌ مُضيئة أو بدرٌ فضي يجريان على مسار القوس بدقة
- **مؤشرات الصلاة** — نقاط تميّز أوقات الصلوات على القوس مع نبضة تنبيه للصلاة القادمة
- **150 نجمة إجرائية** — توليد برمجي حقيقي لا نمط CSS متكرر

### 🕌 المحتوى المعلوماتي
- **التاريخ الهجري** — يُعرض يومياً باستخدام `Intl.DateTimeFormat` مع التقويم الإسلامي، مُزامَن مع التوقيت المحلي للمدينة
- **مواقيت الصلاة الخمسة** — من [`Aladhan API`](https://aladhan.com) باستخدام طريقة رابطة العالم الإسلامي، مُحسَّبة للتاريخ المحلي الصحيح للمدينة
- **شريط النهار والليل** — مقارنة بصرية فورية مع نص يُحدد أيهما أطول وبكم
- **عداد تنازلي** — "بقي على الغروب/الشروق: HH:MM:SS" يُهيّئك نفسياً للحدث القادم
- **التوقيت القياسي** للمدينة المختارة لحظياً (يدعم 12 و24 ساعة)

### 🌍 المدن والتوسّع
- **مدن ليبية افتراضية** — طبرق، بنغازي، طرابلس
- **إضافة أي مدينة عالمياً** — اكتب الاسم بالإنجليزية، يبحث Nominatim/OpenStreetMap ويُضيفها فوراً
- **حفظ المدن المُضافة** — تُخزَّن في `localStorage` وتظهر عند كل تشغيل
- **مشاركة الرابط** — معامل URL `?city=طبرق` يحفظ المدينة ويُمكّن المشاركة
- **WakeLock API** — يمنع إيقاظ الشاشة أثناء العرض (مثلاً على شاشات المساجد)

## ⚙️ كيف يعمل

### 1. جلب البيانات الفلكية
عند اختيار مدينة، يُطلق التطبيق طلباً واحداً جامعاً لـ Open-Meteo يجلب بيانات الأمس واليوم وغداً:

```javascript
const url = `https://api.open-meteo.com/v1/forecast`
          + `?latitude=${lat}&longitude=${lng}`
          + `&daily=sunrise,sunset,daylight_duration`
          + `&timezone=auto&past_days=1&forecast_days=2`;
```

يعيد الـ API أوقات الشروق والغروب بالتوقيت المحلي للمدينة، مع `utc_offset_seconds`.

### 2. تحويل الأوقات إلى UTC مطلق
الأوقات المُعادة سلاسل نصية بالتوقيت المحلي مثل `"2024-06-15T05:30"`. التحويل:

```javascript
// الخطوة 1: عامِل السلسلة المحلية كـ UTC (أضف Z)
// الخطوة 2: اطرح منها الإزاحة → UTC الحقيقي
const parseLocalAsUTC = (str) =>
    new Date(str + 'Z').getTime() - (offMins * 60_000);
```

مثال: شروق محلي `05:30` في مدينة UTC+2 (120 دقيقة):
- `new Date("…T05:30Z").getTime()` = 05:30 UTC بالميلي ثانية
- `− 120 × 60_000` = 03:30 UTC ← الشروق الحقيقي بالـ UTC ✓

### 3. حساب الساعة التكيفية
```javascript
// الوقت المنقضي منذ بدء الطور (الشروق أو الغروب)
const elapsedMs = Date.now() - phaseStartMs;

// الساعة الحالية (1-indexed)
const swahiliHour = Math.floor(elapsedMs / 3_600_000) + 1;
```

لا حدّ أقصى عند 12 — يعكس الرقم الطول الفعلي للنهار أو الليل في مدينتك.

### 4. تحريك الجسم السماوي على القوس
```javascript
// progress يتراجع من 0 (شروق/غروب) إلى 1 (غروب/شروق التالي)
const angle = Math.PI * (1 - progress);

// المركز: (150, 140) — الشعاع: 130 — sweep-flag=1 (عكس عقارب الساعة في SVG y-down)
const cx = 150 + 130 * Math.cos(angle); // أفقياً
const cy = 140 - 130 * Math.sin(angle); // الطرح → فوق الأفق (y يزداد للأسفل)
```

| `progress` | `angle` | الموضع | الحدث |
|:---:|:---:|:---:|:---:|
| 0.00 | π | (20, 140) | الشروق / الغروب |
| 0.25 | 3π/4 | (58, 48) | الضحى / أوائل الليل |
| 0.50 | π/2 | (150, 10) | الذروة |
| 0.75 | π/4 | (242, 48) | العصر / متأخر الليل |
| 1.00 | 0 | (280, 140) | الغروب / الشروق |

### 5. تدرج لون السماء الديناميكي
كل ثانية تُحسب درجة لونية مُدمجة بين 5 محطات بدالة `lerpHex`:

```
شروق   → صباح   → ظهر    → عصر    → غروب
#FEF3C7  #E0F2FE  #F0F9FF  #FEF9C3  #FDE68A   (اللون العلوي)
#FDBA74  #BAE6FD  #E5E7EB  #FDE68A  #FDBA74   (اللون السفلي)
```

لا يتطلب المشروع أي تثبيت أو أدوات بناء:


## 🎨 مبادئ التصميم

### الفلسفة البصرية — "السماء الحية"
صُمِّمت الواجهة على مبدأ أن **الشاشة هي السماء نفسها**:

- **الخلفية** = السماء الفعلية بألوانها المتغيّرة كل ثانية
- **القوس** = مسار الشمس/القمر فوق الأفق
- **النجوم** = تظهر تدريجياً ليلاً فقط
- **التوهّج** = ضوء الشمس والقمر عبر SVG filter

### الجانب النفسي

| المبدأ | التطبيق |
|---|---|
| **الربط بالطبيعة** | الساعة تبدأ من الشروق لا منتصف الليل |
| **التقدم المرئي** | القوس يمتلئ من اليسار — إشباع بصري لحظي |
| **الاستباق الذهني** | العدّاد التنازلي يُهيّئك للغروب/الشروق |
| **المعنى الروحي** | نقاط الصلاة تربط الوقت بالعبادة |
| **التباين الموسمي** | يُذكّرك متى يتساوى الليل والنهار |

### إمكانية الوصول (Accessibility)
- `aria-label` على جميع الأزرار التفاعلية
- `aria-live="polite"` على الساعة والعداد والطور
- `aria-atomic="true"` على مجموعات العرض الزمني
- `role="alert"` على رسائل الخطأ
- `role="status"` على شاشة التحميل
- `sr-only` label على حقل الإدخال
- تباين ألوان > 4.5:1 (WCAG AA) في جميع الثيمات

## 📱 مشاركة الرابط

يدعم التطبيق معامل URL `?city=` لمشاركة مدينة بعينها:

```
https://azzubairx.github.io/SolarisSwahili/?city=طبرق
https://azzubairx.github.io/SolarisSwahili/?city=Cairo
https://azzubairx.github.io/SolarisSwahili/?city=Istanbul
```

عند فتح الرابط يُحمَّل التطبيق مباشرةً على المدينة المحددة. إن كانت المدينة غير موجودة في القائمة المحفوظة، يبحث التطبيق عنها تلقائياً عبر Nominatim.
